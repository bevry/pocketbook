## Install

### [Node](http://nodejs.org/), [Browserify](http://browserify.org/)
- Use: `require('pocketbook')`
- Install: `npm install --save pocketbook`

### [Ender](http://ender.jit.su/)
- Use: `require('pocketbook')`
- Install: `ender add pocketbook`
